# Requirements
- neovim 0.5+

## Installation
- create python venv
```
python -m venv ~/.vim/python3
source ~/.vim/python3/bin/activate
pip install -r ~/dotfiles/requirements.txt
```

## other dependencies
- ripgrep
- bat
- fd
- clangd
- wsl-open(if wsl)
- ~/.cache/skk-jisyo
- SKK-JISYO.L(windows: ~/AppData/Local/skk/SKK-JISYO.L, linux: /usr/local/share/skk/SKK-JISYO.L)
